ArCode

Arcode adalah sebuah platform CLI (Command Line Interface) yang dibuat menggunakan bahasa pemrograman C++. Platform ini dirancang agar pengguna dapat menjalankan berbagai game dalam satu platform CLI. Pada platform ArCode sendiri terdapat 3 game yang dapat dimainkan seperti Guess The Word, Typing Game Delux, dan Eye of Cthulu.

FunWorld

FunWorld adalah sebuah sistem informasi CLI (Command Line Interface) yang dibuat untuk mensimulasikan sistem informasi taman bermain menggunakan bahasa pemrograman Java. Pada sistem informasi FunWorld terdapat 2 peran, yaitu sebagai Admin dan User. Admin akan berperan untuk mengelola taman bermain seperti kelola tiket, wahana bermain, user, dan barang. Kemudian untuk User, User akan berperan sebagai pengunjung taman bermain tersebut dan akan bisa melakukan berbagai hal di taman bermain tersebtu seperti melihat daftar wahana bermain, pesan tiket, titip barang, dll.

Fotografi Sejahtera

Fotografi sejahtera adalah sebuah website sederhana yang dibuat menggunakan HTML, CSS, JS, dan PHP. Pada website Fotografi Sejahtera terdapat 2 peran yaitu sebagai Admin dan User. Peran Admin adalah untuk mengelola website seperti mengelola paket yang ada pada website, mengelola pesan yang masuk dari user, dan mengelola galeri foto yang akan ditampilkan. Sedangkan untuk User, User akan dapat melihat paket, melihat galeri, memesan paket, melihat testimoni pelanggan, dan juga mengirim pesan kepada Admin.
